# Advanced Markdown

## Notes (Admonitions)

### Defaults

:::note
I am a Note
:::

:::tip
I am a tip
:::

:::caution
I am caution
:::

:::danger
I am Danger
:::

### Custom Note

There are two ways to customize the Notes

- extending the integration

- using Note Component
